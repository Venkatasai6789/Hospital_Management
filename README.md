# MediConnect - Managed Healthcare System

MediConnect is a professional hospital management system designed to streamline interactions between patients, doctors, and administrators. Built with React, Vite, and Node.js.

## Modules

### 👑 Administrator
- Manage departments and personnel.
- Oversee patient data and hospital resources.
- Detailed analytics and reporting.

### ⚕️ Doctor
- View and manage patient appointments.
- Access patient medical history and lab reports.
- Real-time interaction and diagnosis tracking.

### 👤 Patient
- Book and manage appointments.
- View personal medical history and test results.
- Intuitive dashboard for health monitoring.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Lucide Icons.
- **Backend**: Node.js, Express, Supabase.
- **Charts**: Recharts.

## Setup Instructions

### Prerequisites
- Node.js (v18+)
- npm

### 1. Installation
Clone the repository:
```bash
git clone https://github.com/Venkatasai6789/Hospital_Management.git
cd Hospital_Management
npm install
```

### 2. Environment Setup
Create a `.env` file in the root and backend directories with your Supabase and API credentials.

### 3. Run the Backend
```bash
cd backend
npm install
npm start
```

### 4. Run the Frontend
```bash
# Return to root
npm run dev
```
